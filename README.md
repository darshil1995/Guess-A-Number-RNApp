# Guess-A-Number-RNApp
A react Native application which takes the user input as a number and randomly generates the number in the second screen. It will check whether the number is correct guessed or not.
<b>Features</b><br>
- Takes the user input as a number<br>
- Generates a random number for a guess and display it in second screen.<br>
- Checks whether the user number is equal to the guessed number.<br>
- If the user number is not correct the user can say whether their number is greater or lower than it, bby clicking the greater and lower buttons. It also displays how much rounds it took to guess the correct number.<br>

<p>
Use of Hooks=> useState, useEffect<br>
use of Custom components=> Buttons, Text, TextInput<br>
Navigations from one screen to other=> StartScreen, GameScreen & GameOverScreen<br>
Validations, Displaying images and many more
